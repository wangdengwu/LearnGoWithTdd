### 指针和错误

#### 需求

1. 实现一个简单的钱包(Wallet)管理功能
	* 可以给钱包存钱Deposit
	* 可以从钱包取钱Withdraw
	* 如果取钱时，资金Balance不够，则报错
2. 实现年龄管理功能
	* 对每个人的年龄+1
	* 以slice的方式传递参数
	* 以map的方式传递参数

#### 总结

1. 值传递
2. 如果需要重用结构体，则可以传递指针，避免copy导致的性能和数据不一致
3. 支持多返回值，方法一般最后一个参数为error
4. 对map的value修改需要重新赋值
